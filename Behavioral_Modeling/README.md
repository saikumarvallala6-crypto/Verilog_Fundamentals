# Behavioral Modeling

This directory contains Verilog HDL examples implemented using **Behavioral Modeling**.

Behavioral modeling describes the functionality of hardware using procedural blocks such as `always` and `initial`. It focuses on system behavior instead of gate-level implementation.

## Topics Covered

- Logic Gates
- Counters
- Flip-Flops
- Flip-Flop Conversions
- Finite State Machines (FSM)
- Shift Registers
- Parameterized FIFO
- Parameterized RAM
- BCD Adder

# Dataflow Modeling

This directory contains Verilog HDL examples implemented using **Dataflow Modeling**.

Dataflow modeling represents hardware using continuous assignments and Boolean expressions through the `assign` statement.

## Topics Covered

- Logic Gates
- Multiplexers
- Demultiplexers
- Encoders
- Decoders
- Comparators
- Binary ↔ Gray Converter
- Full Adder
- Half Adder
- Full Subtractor
- Half Subtractor
- Ripple Carry Adder
- Carry Look Ahead Adder
- Priority Encoder
- Parity Generator
- Parity Checker
- Tristate Gates
- Multipliers

# Logic Gates

## Description

This example demonstrates the implementation of basic logic gates using Verilog gate-level primitives.

## Gates Implemented

- AND
- OR
- NAND
- NOR
- XOR
- XNOR
- NOT
- BUFFER

## File Structure

The Verilog source file contains:
- Design Under Test (DUT)
- Testbench (TB)

## Modeling Style

Gate-Level Modeling

## File

logic_gates.v

## Simulation

The source file contains both the DUT and testbench.

## Software

- QuestaSim
- ModelSim

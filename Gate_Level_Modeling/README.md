# Gate Level Modeling

This directory contains Verilog HDL implementations of common digital circuits using **gate-level modeling**.

Gate-level modeling describes digital circuits using Verilog primitive gates such as:

- and
- or
- xor
- xnor
- nand
- nor
- not
- buf

Each example contains both the DUT and the testbench in a single Verilog source file for easy simulation and learning.

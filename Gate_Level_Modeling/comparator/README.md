# Comparator

## Overview
Implementation of a 1-bit Comparator using Verilog gate-level modeling.

## Outputs
- A > B
- A < B
- A == B

## Modeling Style
Gate-Level Modeling

## File Structure
The Verilog source file contains:
- Design Under Test (DUT)
- Testbench (TB)

## File
comparator.v

## Simulation
The source file includes both the DUT and the testbench.

## Tools Used
- Verilog HDL
- QuestaSim / ModelSim

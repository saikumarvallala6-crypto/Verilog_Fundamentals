# 2x4 Decoder

## Overview
Implementation of a 2-to-4 Decoder using Verilog gate-level modeling.

## Modeling Style
Gate-Level Modeling

## File Structure

The Verilog source file contains:
- Design Under Test (DUT)
- Testbench (TB)

## File
decoder_2x4.v

## Simulation
The source file includes both the DUT and the testbench.

## Tools Used
- Verilog HDL
- QuestaSim / ModelSim

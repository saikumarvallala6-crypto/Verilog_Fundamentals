# Even Parity Checker

## Overview
Implementation of an Even Parity Checker using Verilog gate-level modeling.

## Modeling Style
Gate-Level Modeling

## File Structure

The Verilog source file contains:
- Design Under Test (DUT)
- Testbench (TB)

## File
even_parity_checker.v

## Simulation
The source file includes both the DUT and the testbench.

## Tools Used
- Verilog HDL
- QuestaSim / ModelSim

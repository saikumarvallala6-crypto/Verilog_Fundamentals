# Full Subtractor

## Overview
Implementation of a 1-bit Full Subtractor using Verilog gate-level modeling.

## Inputs
- A
- B
- Bin

## Outputs
- Difference
- Borrow

## Modeling Style
Gate-Level Modeling

## File Structure

The Verilog source file contains:
- Design Under Test (DUT)
- Testbench (TB)

## File
full_subtractor.v

## Simulation
The source file includes both the DUT and the testbench.

## Tools Used
- Verilog HDL
- QuestaSim / ModelSim

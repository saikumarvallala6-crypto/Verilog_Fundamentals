# Gray to Binary Converter

## Overview
Implementation of a Gray Code to Binary Converter using Verilog gate-level modeling.

## Modeling Style
Gate-Level Modeling

## File Structure

The Verilog source file contains:
- Design Under Test (DUT)
- Testbench (TB)

## File
gray_to_binary.v

## Simulation
The source file includes both the DUT and the testbench.

## Tools Used
- Verilog HDL
- QuestaSim / ModelSim

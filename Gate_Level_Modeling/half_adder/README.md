# Half Adder

## Overview
Implementation of a 1-bit Half Adder using Verilog gate-level modeling.

## Inputs
- A
- B

## Outputs
- Sum
- Carry

## Modeling Style
Gate-Level Modeling

## File Structure

The Verilog source file contains:
- Design Under Test (DUT)
- Testbench (TB)

## File
half_adder.v

## Simulation
The source file includes both the DUT and the testbench.

## Tools Used
- Verilog HDL
- QuestaSim / ModelSim

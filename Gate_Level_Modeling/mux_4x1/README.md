# 4x1 Multiplexer

## Overview
Implementation of a 4-to-1 Multiplexer using Verilog gate-level modeling.

## Modeling Style
Gate-Level Modeling

## File Structure

The Verilog source file contains:
- Design Under Test (DUT)
- Testbench (TB)

## File
mux_4x1.v

## Simulation
The source file includes both the DUT and the testbench.

## Tools Used
- Verilog HDL
- QuestaSim / ModelSim

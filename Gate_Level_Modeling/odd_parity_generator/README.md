# Odd Parity Generator

## Overview
Implementation of an Odd Parity Generator using Verilog gate-level modeling.

## Modeling Style
Gate-Level Modeling

## File Structure

The Verilog source file contains:
- Design Under Test (DUT)
- Testbench (TB)

## File
odd_parity_generator.v

## Simulation
The source file includes both the DUT and the testbench.

## Tools Used
- Verilog HDL
- QuestaSim / ModelSim

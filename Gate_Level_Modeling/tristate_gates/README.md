# Tristate Gates

## Overview
Implementation of tri-state buffers using Verilog gate-level primitives.

## Features
- High-Impedance (Z) Output
- Enable Control

## Modeling Style
Gate-Level Modeling

## File Structure

The Verilog source file contains:
- Design Under Test (DUT)
- Testbench (TB)

## File
tristate_gates.v

## Simulation
The source file includes both the DUT and the testbench.

## Tools Used
- Verilog HDL
- QuestaSim / ModelSim

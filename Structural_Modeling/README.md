# Structural Modeling

This directory contains Verilog HDL examples implemented using **Structural Modeling**.

Structural modeling builds complex digital circuits by interconnecting lower-level modules.

## Topics Covered

- Logic Gates using NAND Gates
- Logic Gates using NOR Gates
- Logic Gates using Decoder
- Logic Gates using Multiplexer
- Half Adder using Half Subtractor
- Full Adder using Half Adders
- Full Subtractor using Half Subtractors
- Ripple Carry Adder
- Carry Look Ahead Adder
- 4×1 MUX using 2×1 MUX
- 8×1 MUX using 4×1 and 2×1 MUX
- 16×1 MUX using 4×1 and 2×1 MUX
- 13×1 MUX using 2×1 MUX
